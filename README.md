Acheived with reverse engineering.</br>
<strong>Works with X11 and Wayland.</strong></br>

# Installation process:</br>
1.) Extract `kuromi-cursor.tgz` inside `$HOME/.icons`</br>
2.) Select your new cursor theme</br>
3.) Enjoy! >w< </br>
